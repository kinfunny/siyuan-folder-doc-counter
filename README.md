# 思源文件夹文档计数

一个很小的思源笔记插件：在左侧文档树中，为笔记本和有子文档的节点显示其下文档数量。

## 功能

- 在文档树标题右侧显示数量
- 支持一级笔记本数量显示
- 默认统计所有层级的子文档，不包含当前文档
- 文档树发生变化时自动刷新
- 插件卸载时自动清理界面上的计数徽标
- 支持设置中切换背景圈显示

## 设置

插件设置中提供“显示背景圈”开关。关闭时只显示纯数字，开启后显示圆角背景。

## 兼容性

已在 SiYuan 3.6.5 桌面端测试。

## 开发

```bash
npm install
npm run dev
```

## 打包

```bash
npm run build
```

将构建产物和 `plugin.json`、`README.md`、`icon.png`、`preview.png` 放入思源工作空间：

```text
<workspace>/data/plugins/folder-doc-counter
```

然后在思源中启用插件。

## 后续可改方向

- 改成只统计直接子文档
- 给不同数量区间设置不同颜色
