# 思源文件夹文档计数

这是一个思源笔记插件，用于在左侧文档树中显示笔记本和子文档数量。

## 功能

- 在文档树中显示数量
- 支持一级笔记本数量显示
- 只有有子文档时才显示
- 默认纯数字
- 可在设置中开启背景圈

## 设置

插件设置中提供“显示背景圈”开关。关闭时只显示纯数字，开启后显示圆角背景。

## 兼容性

已在 SiYuan 3.6.5 桌面端测试。

## 安装

将 `index.js`、`index.css`、`plugin.json`、`README.md`、`README_zh_CN.md`、`icon.png`、`preview.png` 放入：

```text
<workspace>/data/plugins/siyuan-folder-doc-counter
```

然后在思源插件列表中启用。
